# Find a way to get a shell on SSH

**Task 1:** **What is the user:password of the admin panel?**

admin:xavier

viewing the source page of the admin web page we find a comment. So now we know that the username is “admin”

To find the admin password, the hint suggests that we use hydra a password cracking tool.  The command used is: **hydra -l admin -P rockyou.txt MACHINE_IP

Upon clicking the RSA private key link, we get this:
![image.png](/assets/images/BruteIt/rsa-private-key.png)

Create a file called id_rsa using a text editor of your choise, in my case I used nano. Copy the content of the RSA private key from the web page to the text editor so that we have it locally.

To decrypt the private key and get the passphrase, we first need to convert it to a form John can understand it, then run John againsta it. The format in question here is a hash format using the code:

comment: Convert the private key to a hash format John can understand
code: ssh2john id_rsa > id_rsa.hash

ssh2john is a python script that comes with John the Ripper. if you dont have it, install using the code:
comment: On Kali Linux (usually pre-installed):
code: sudo apt update && sudo apt install john -y

comment; If ssh2john isn't available, you can get it directly:
wget https://raw.githubusercontent.com/openwall/john/bleeding-jumbo/run/ssh2john.py
python3 ssh2john.py id_rsa > id_rsa.hash
